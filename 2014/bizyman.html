Workflow

[Step 1] Create Report

  • With smartphone or from webbrowser
  • Attach photo and description
  • Provide additional information,
    ie. severity, classification, etc.

[Step 2] Review Report

  • Change report status
  • Assign report to group or person
  • Set additional attributes,
    ie. tags, start or end date, etc.
  • Document the process

[Step 3] Resolve Report

  • Handle the report and resolve the task
  • Change report status (in progress, resolved)
  • Document work (ie. photos, working time)
  • Pass work to new person (optional)

[Step 4] Close Report

  • Verify the result
  • Change report status to "closed"
  • Document the process
