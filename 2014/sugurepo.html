Location-based Task Management for Mobile Business

Created by Daniel Kastl / @dkastl

The Idea

「すぐレポ」is a Location-based Task Management Solution for Mobile Business and Asset Maintenance:

  • Facilitates On-site Customer Services
  • Optimizes Business Processes
  • Integrates Service Workflows
  • Fosters Mobility and Service Quality

For any mobile work!

  • Courier, care services
  • On-site field services
  • Infrastructure works
  • Any work with local tasks ...

=> Location-based Task Management

Workflow

[Step 1] Create Report

  • With smartphone or from webbrowser
  • Attach photo and description
  • Provide additional information,
    ie. severity, classification, etc.

[Step 2] Review Report

  • Change report status
  • Assign report to group or person
  • Set additional attributes,
    ie. tags, start or end date, etc.
  • Document the process

[Step 3] Resolve Report

  • Handle the report and resolve the task
  • Change report status (in progress, resolved)
  • Document work (ie. photos, working time)
  • Pass work to new person (otional)

[Step 4] Close Report

  • Verify the result
  • Change report status to "closed"
  • Document the process
